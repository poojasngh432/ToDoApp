# Android To-Do App

## Stack
- Kotlin
- XML Views
- Room DB
- MVVM + Clean Architecture
- Coroutines + Flow
- JUnit, Espresso

## Architecture
UI → ViewModel → UseCases → Repository → Room

## Run App
1. Clone repo
2. Open in Android Studio
3. Run on emulator/device

## Tradeoffs
- No Compose (XML chosen for simplicity)
- Basic filtering logic in ViewModel

## Improvements
- Paging
- Jetpack Compose migration
- Material 3 theming
- WorkManager for reminders
